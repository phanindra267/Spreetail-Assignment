## Overview

A shared expenses app built for flatmates to track expenses, split costs, and settle debts!

## Features
1. Login module
2. Create/manage groups with changing membership over time
3. Create/manage expenses with multiple split types
4. View group-wise and individual balance summaries
5. Settle debts
6. Import expenses_export.csv with anomaly detection
7. Relational database (PostgreSQL)

---

## Setup Instructions

### Backend Setup
1. Navigate to backend directory
2. Install dependencies: `npm install`
3. Set up environment variables in .env
4. Run database migrations
5. Start backend server: `npm start`

### Frontend Setup
1. Navigate to frontend directory
2. Install dependencies: `npm install`
3. Start frontend: `npm start`
